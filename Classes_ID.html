Estilo CSS aplicado a Classes e IDs

DEFINIÇÃO: ID

ID são identificadores, em uma comparação imagine um ônibus com vários lugares, a definição de ID se aplicaria aqui como sendo cada lugar, identificando um PONTO ÚNICO e EXCLUSIVO.

A definição de ID dá-se através de um # + NomeDaVariavel {atributos;}

Usamos o ID uma única vez (lembrar da definição) dentro da página, ou seja, uma vez por página. Também usa o ID em JS, aqui usamos como estruturação.

USOS: No ID poderiamos usar no HEADER e Classes nas várias colunas na estrutura de um site.

DEFINIÇÃO: CLASSES

Classes podem ser usadas diversas vezes, na comparação do ônibus. Classificar um passageiro de acordo com o sexo, assim podemos ter uma ou mais pessoas do sexo 'masculino' ou 'feminino'. Podendo definir VÁRIOS ITENS dentro de um contexto. Normalmente quando formatamos usamos as Classes para formatar as EXCEÇÕES.

A definição de classes dá-se através de um '.' (ponto) + NomeDaVariavel {atributos;}

Lembrando da definição de Classes e ID

As formatações feitas no são formtações gerais. Normalmente quando formatamos usamos as Classes para formatar as EXCEÇÕES.
