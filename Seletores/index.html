Seletor de Filho

Seleciona um elemento que é filho direto de outro. O sinal de ">" lê-se "filho de:". Exemplo: "li>a", ou seja, um link que é filho de "li". A leitura é de trás para frente, ou seja, do final. Seletor de irmão adjacente

Seleciona um elemento que é o irmão próximo de outro. Exemplo: "h3+p {}", assim selecionamos o primeiro elemento "p" depois de quaisquer elemento "h3", mas não outros elementos "p".

Seletor de irmão geral

Seleciona um elemento que é um irmão de outro, embora ele não precise ser o elemento diretamente precedente. MAIS APOS O ELEMENTO ESCOLHIDO. Exemplo: "h3~p {}", se houvesse dois elementos "p" que fosse irmãos de um elemento "h1", essa regra se aplicaria ao dois. ASSIM, SELECIONA O IRMÃO APOS O "H3".
