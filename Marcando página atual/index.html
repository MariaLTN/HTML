Barra de Navegação Vertical - Marcação da Página Atual

Uma complementação a anotação anterior ("Barra de Navegação Vertical"), é para caso algum elemento não funcione corretamente, ainda consigamos ver e analisar o menu. Por exemplo, se a imagem não carregar temos uma cor para aparecer.

Para marcar a página que estamos, podemos colocar o "a:hover" uma classe que indica que cada vez que o link for selecionado/ciclado usaremos aquela formatação. Assim ficando, "a:hover .selecionado a" Principalmente usado quando temos uma linguagem de programação atrelado. Assim, colocamos no "li class= "selecionado""

Outra abordagem é com o uso dos Seletores, selecionando as páginas de forma automática. É preciso criar uma estrutura. O id="Home" indica que estamos na página HOME e class="Home" também. E o mesmo é aplicado para as páginas "Dentista" (Id e class) e com "Dermatologista".

#Home .Home a, #Dentista .Dentista a, #Dermatologista .Dermatologista a{ Background-position: right bottom; background-color: #666FD7; }

O uso do "id=Navegação" é pq podem existir em uma página mais de um tipo de navegação. Assim, torna mais especifico ainda e seleciona o elemento certo.
