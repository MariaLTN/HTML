O "z-index" indica qual elemento aparece primeiro. Por padrão, primeiro teremos o "Conteudo 01" e por cima temos o "Conteudo 02". Para alterar isso, é possível ao usarmos o "z-index: número;" E quanto maior o número, mais a frente ele aparece.

Para fazer o "Conteudo 02" ficar no fundo e o "Conteudo 01" na frente podemos usar "z-index:1; (na classe css Conteudo01)" e "z-index:0; (na classe css Conteudo02)". Se os "z-index" de todos os elementos forem iguais, eles seguem a ordem do documento, ou seja, Conteudo 01 depois Conteudo 02 e Conteudo 03.

O "z-index" também é chamado de CONTEXTO DE EMPILHAMENTO.
