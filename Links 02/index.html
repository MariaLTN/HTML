Os links tem ESTADOS, ele se apresenta com uma cor quando o link não foi visitado e uma outra cor quando o link foi visitado.

GOOGLE | LOREM | GITHUB | CORES

Para os links não visitados podemos formatar com "a:link {formatação}"; Usado a cor LARANJA.

Para os link visitados usamos "a:visited{formatação};" Usado aqui a cor ROSA;

Temos ainda o link ativo. A cor muda quando clicamos no link. Podemos formatar com "a:active {formatação}";Nesse caso o VERDE;

E por fim, temos uma ação quando passamos o cursor sobre um link. Nesse caso usamos, "a:hover {formatação}"; Nesse caso o AZUL;

Assim temos 4 ações que podemos formatar em relação aos links: Link (Não visitados), Visited (Visitados), Active (Clicamos) e Houver (Passamos o cursor). Além das formatações básicas que podem ser feitas, como o color, background...

Normalmente não se formata os link já visitados, pois isso é feito de outra maneira. Como por exemplo, pode-se informar qual página o usuário está apenas utilizando CSS. Então a cor do "link" e "visited" com por exemplo a mesma coloração. Assim no "style.css" posso ter "a:link, a:visited {color: green}" por exemplo;

Embora ao usar essas 4 ações é legal que eles se apresentem nessa ordem: link, visited, hover & active
