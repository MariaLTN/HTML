Barra de Navegação Vertical

Para a criação de uma Barra de Nvaegação Vertical precisamos customizar uma lista. Assim, remover os "*" (pontos de marcação de itens) da lista é o primeiro passo. A exclusão pode ser feita com o "list-style-type: none". Ao usarmos listas, vermos além dos pontos de marcação de itens, vemos também um espaçamento sendo aplicado por padrão, que pode ser removida com o "margin:0" e "padding:0".

Mas, para a criação de uma Barra que levem a outras páginas precisamos usar o "a href="" ". E precisamos customizá-los também, como em uma aplicação maior podem existir outros usos de "a href=""", temos que referenciar o "a" certo. Usando conhecimento das anotações passadas SELETORES, podemos usar um seletor que indique que queremos usar o "a" que está dentro de uma "ul" (Lista não ordenada). Assim, após essa referencia feita. A estilização desse elemento "ul a {}" é possível, de acordo com a imagem usada de "width: 200px" e Height: 40px e com as duas cores da imagem podemos criar um efeito de "a:hover" para alterar o fundo da imagem conforme o usuário passa o mouse por cima da barra. Logo, será violeta quando o cursor não está em cima e Azul quando está.

Os elementos de uma "ul" são naturalmente um elemento inline (o tamanho varia de acordo com o conteúdo), usamos o "display: block" para que os elementos fiquem o tamanho fixo. Colocamo uma "color: black" e "text-decoration: none" para retirar o sublinhado das "a" e colocarmos com uma cor preta. Como fundo colocamos o "background: url(Imagens/Menu.png) no-repeat"

Outro ponto importante para a formatação da Barra é o uso do "line-height: 40px" e "text-indent: 40px". O primeiro garante que que o alinhamento da fonte (Por exemplo, Home) fique localizada no meio do seu bloco; O segundo garante o posicionamento do texto a 40px (A partir da esquerda).

CONTEÚDO IMPORTANTE SOBRE LINE HEIGHT & FONT SIZE/

Para a movimentação do cursor, temos o uso do "a:hover", como usaremos uma outra parte da imagem do background (Parte laranja). Assim, para garantir que essa parte aparece apenas usamos o "background-position: right bottom"
